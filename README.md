# 🎨 LydiaArt - Interior & Architecture Design Studio
A sleek, modern, and responsive website for LydiaArt, a boutique interior and architecture design studio. Built to showcase the team's portfolio, philosophy, and creative process.
## https://via.placeholder.com/800x400?text=LydiaArt+Website+Preview
# ✨ Features
Responsive Design: Looks beautiful on desktops, tablets, and mobile devices.

Modern Aesthetic: Clean layout with a soft color palette that reflects a design studio's brand.

Team Introductions: A dedicated "Meet the Team" section featuring embedded video introductions.

Portfolio Gallery: A place to showcase stunning design projects (ready for your content).

Contact Form: For potential clients to easily get in touch.

# 🛠️ Built With
HTML5 - For semantic and accessible structure.

CSS3 - For styling, animations, and responsive layout.

Git & GitHub - For version control and deployment.
