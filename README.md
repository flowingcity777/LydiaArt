# Lydia Art

Lydia Art is a personal website created to preserve and organize artwork from different periods of my life. This space gathers drawings, paintings, and handmade pieces into a quiet, accessible archive.

## Purpose

This site is not a commercial portfolio.
It is a personal collection meant to:

* Preserve artwork over time
* Organize pieces in a clear and simple structure
* Share meaningful works with family

## Structure

The homepage is organized into three main sections:

* **Featured Works** — A small selection that introduces the tone of the collection
* **Selected Works** — Additional pieces, including drawings and handmade works
* **About** — The purpose and intention behind the site

## Design Approach

The design focuses on:

* Simplicity and clarity
* Consistent layout and spacing
* Respecting the original artwork without distortion
* A calm and minimal visual presentation

## Technologies Used

* HTML
* CSS
* Responsive layout (CSS Grid)
